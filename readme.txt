This is advanced version of my drop in image slideshow gallery. In this gallery each image is dropped into view. Slideshow will pause on mouse over. This plugin have separate image management link in the admin dashboard. From that dashboard link admin can manage the gallery images and target links. In the old drop in image slideshow plugin the images retrieved directly from folder.

* Free.
* Easy to customize.
* Support all browser.
* Automatically pauses on mouse over.
* Short code available for pages and posts.
* Hyperlink option to each images.
* Display image order option available also random order option available.

We have three option to install and configure this wp image slideshow plugin. 

http://www.gopiplus.com/work/2011/05/06/wordpress-plugin-wp-image-slideshow/
